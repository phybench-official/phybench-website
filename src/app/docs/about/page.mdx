## 上传题目

### 选择题目

登录后，你可以在 **提交** 页面上传题目。在选择了心仪的题目后，你可能需要对其进行适度的调整或删减，以符合格式要求。

在当前阶段，我们希望题目满足以下要求：

1. 不含多个小问，**只有一个小问**；对于含有多个小问的题目，你可以删除其他小问，保留必要的解题逻辑，只留下最关键的小问；
2. 答案应为 **简单的表达式**，不涉及文字叙述、绘图描述或需要分类讨论的复杂表达式；
3. **答案中涉及的所有符号必须在题干中显式给出**；原有题干中常常忽略的气体常数 $R$、光速 $c$ 等，你可能需要手动补充；
4. 题干 **不得包含图片**；如果题干确实含有图片，你可以尝试使用图文生成模型等方式，概括图片的含义并融入题干的文字表述中。

此外，我们希望 **题目不能被当前的优秀 AI 轻易攻破** 。尽管 AI 表现并非必填项，但如果你选择 **Deepseek R1**、**o3-mini** 或 **Gemini** 回答出错的题目，并在后续步骤中**上传出错的 AI 表现**，题目将**更容易通过审核**。

### 题目解答与答案

在 **提交** 页面的 **上传题目** 栏，我们需要提供 **Markdown 格式的解题步骤** ；在此推荐一种简易的解决办法：**使用 MinerU 等文档提取工具，自动将 PDF 文件转换为 Markdown 格式** 。

请注意：如果解题步骤包含图片，MinerU 等文档提取工具通常会在其服务器存储图片，并提供图片的 url 字符串；因此， **这一办法对含图的解题步骤同样适用，并且无须额外操作** 。

<img src="https://img-api.pku3d.com/ota/parkcai/vghfR4PR.png" alt="图4" width="1000"/>

MinerU 等文档提取工具的转换正确率并非 100%，但解题步骤部分主要是给审核员看的，厘清大致思路、陈列大部分公式即可，不必求全责备。

上传解题步骤与题目答案后，还需要设置题目变量（符号），并标注符号范围。在此作两点说明如下：

1. **符号范围不是题目本身包含的信息，是为方便后续数值验证而人为引入的界限**；你可以在满足自然约束（如非负性）的前提下任意框定一个范围，但不必过分极端；
2. 如果符号是一个常量（如气体常数$R$），请在最小值和最大值输入相同的值。

<img src="https://img-api.pku3d.com/ota/parkcai/4mBiGEjm.png" alt="图5" width="1000"/>

## 积分与奖励

若题目通过审核，审核员会依据题目质量给予积分奖励。

在即将开放的网站功能中，你可以在**发现**页面上查看积分榜。积分代表劳动时长，可以**替代劳动教育课时**（与 CUPT 类似）、**兑换志愿课时**、**兑换费米子文创等精美小礼品**。如果你的积分足够高，你会被列为**PhyBench 工作的共同作者（Author）**。
